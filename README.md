# weather
## 環境依賴
- Python 3.X

## 功能描述
WeatherHK 是命令行界面，通過調用香港天文台的 API 獲取香港本地天氣報告和九天天氣預測。

### 本地天氣報告
本地天氣報告將會返回以下數據：
- 天氣概況
- 特別天氣提示 （如有）
- 熱帶氣旋位置 （如有）
- 溫度
- 雨量 （如有）
- 警告信息 （如有）
- 紫外線指數 （如有）（暫定）
- 閃電 （如有）（暫定）

### 九天天氣預測
九天天氣預測可以返回九天或指定某天的天氣預測，當中包含以下數據：
- 天氣概況
- 風
- 天氣
- 最高溫度
- 最低溫度
- 相對濕度
- 降雨概率
